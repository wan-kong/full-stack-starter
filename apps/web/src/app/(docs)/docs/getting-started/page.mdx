# Getting Started

Learn how to integrate FullStackStarter into your application.

## Prerequisites

- Node.js 18+ or Bun
- A project that needs authentication

## Installation

Install the FullStackStarter SDK in your project:

```bash
bun add @repo/sdk
```

## Configuration

Create an auth client instance:

```ts
import { createAuthClient } from "@repo/sdk";

const auth = createAuthClient({
  baseURL: "https://your-domain.com",
});
```

## Sign In

Use the client to authenticate users:

```ts
const result = await auth.signIn.email({
  email: "user@example.com",
  password: "secure-password",
});

if (result.error) {
  console.error("Sign in failed:", result.error);
} else {
  console.log("Welcome!", result.data.user);
}
```
